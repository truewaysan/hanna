# 🐩 ハンナ お世話手帳

トイプードルの子犬「ハンナ」の毎日のお世話を記録するスマホアプリ（PWA）。

## できること

- **ワンタップ記録**：おしっこ💧 / うんち💩 / ごはん🍚 / 体重⚖️ / くすり💊 / 通院🏥
- **ホーム**：前回からの経過時間がひと目で分かる（間隔が空くと色が変わる）
- **ルーティン**：1日のスケジュール（編集可・毎日0時にチェックリセット）
- **きろく**：体重グラフ、通院/ワクチンの次回予定、種類で絞れる履歴
- **通知**：ルーティン時刻・トイレ間隔のリマインド（要・通知許可）
- **バックアップ**：JSON で書き出し／読み込み

## スマホへの追加（iPhone）

1. このサイトを Safari で開く
2. 共有ボタン → **「ホーム画面に追加」**
3. 初回に「通知を許可する」をタップ

## 仕様メモ

- 記録はブラウザの localStorage に保存（**現状は端末ごとに独立**）
- 家族でリアルタイム共有する版は、データ層 `store.js` をクラウド（Supabase 等）に
  差し替えて対応予定
- 静的ファイルのみ（ビルド不要）。`index.html` を配信するだけで動く
